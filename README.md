# Sistemi Informativi

Materiale didattico per il corso di Sistemi Informativi  
Corso di Laurea in Ingegneria Industriale  
Università Campus Bio-Medico di Roma (UCBM) — A.A. 2025/2026

## Struttura del Repository

- `docs/` — Materiale didattico in PDF per gli studenti
  - `python/` — Ripasso di Python di base
  - `git/` — Materiale introduttivo su Git e GitHub
  - `tabelle/` — Percorso da rappresentazione tabellare a modello relazionale
  - `slides/` — Lucidi delle lezioni
- `labs/` — Esercitazioni di laboratorio
  - `tabelle/` — Esempi Python su logica tabellare e operazioni base
  - `rubrica_oop/` — Progetto CRUD rubrica telefonica (OOP + persistenza JSON)

## Strumenti Utilizzati

- Python 3.12
- Paradigma Object-Oriented
- Persistenza su file JSON
